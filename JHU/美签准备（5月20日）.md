# 核心原则是：
**真实、简洁、自信，并始终围绕“我是去真实学习，且学成后一定回国”这一中心。**

### 二、核心回答策略（贯穿始终）

1. **答案前置**：先给出直接、肯定的结论，再视情况补充一句原因。例如：“毕业后我计划回国。因为...”
    
2. **主动引导**：回答时，可以自然地将话题引向你准备充分的材料。例如，谈到资金时，可以边说边示意手中的存款证明文件。
    
3. **保持一致**：所有口头回答必须与**DS-160表格**、**I-20文件**、以及你提交的所有书面材料信息**100%一致**。
    
4. **积极正面**：即使被问到挑战性问题（如“为什么不去中国读书？”），也不要争论。可以回答：“美国在该领域有领先的研究环境和资源，能帮助我实现学术目标，最终将所学贡献于国内发展。”
    

### 三、临场要点

- **诚实第一**：不知道或没听清就礼貌地说“Pardon?”或“Could you please repeat the question?”，不要猜测或编造。
    
- **材料熟悉**：确保你知道每一份材料放在哪里，被问到时能迅速取出。
    
- **心态放松**：将面签视为一次简短的留学面试，签证官只是例行核实信息。保持微笑、眼神交流、表达清晰。
