## 习题 1: 一阶温度传感器 — 两次阶跃响应

### 题目

> 一个温度测量系统由线性元件组成，整体稳态灵敏度为 1。系统的动态特性由传感元件的一阶传递函数决定。在 $t=0$ 时，传感器从 20°C 的空气中突然转移到沸水中。一分钟后，传感器突然转移回空气中。计算以下时刻的系统动态误差：$t = 10, 20, 50, 120, 300$ s。

| 传感器参数 | 数值 |
|-----------|------|
| 质量 $m$ | $5 \times 10^{-2}$ kg |
| 表面积 $A$ | $10^{-3}$ m² |
| 比热 $C$ | 0.2 J·kg⁻¹·°C⁻¹ |
| 空气传热系数 $h_a$ | 0.2 W·m⁻²·°C⁻¹ |
| 水传热系数 $h_w$ | 1.0 W·m⁻²·°C⁻¹ |

### 解答

#### Step 1: 计算两个介质中的时间常数

$$\tau = \frac{mC}{UA} = \frac{mC}{hA}$$

**水中**：
$$\tau_w = \frac{5 \times 10^{-2} \times 0.2}{1.0 \times 10^{-3}} = \frac{0.01}{0.001} = \boxed{10 \text{ s}}$$

**空气中**：
$$\tau_a = \frac{5 \times 10^{-2} \times 0.2}{0.2 \times 10^{-3}} = \frac{0.01}{0.0002} = \boxed{50 \text{ s}}$$

#### Step 2: 建立温度响应方程

> ⚠️ **关键**: 动态误差 $E(t) = T_m(t) - T_{true}$，其中 $T_m$ 是测量值，$T_{true}$ 是当前真实环境温度。

**阶段 1 — 在沸水中 (0 ≤ t ≤ 60 s)**：
- 初始温度 $T_m(0) = 20°C$
- 环境温度 $T_{\infty,1} = 100°C$
- $$T_m(t) = 100 + (20 - 100)e^{-t/10} = 100 - 80e^{-t/10}$$
- **动态误差**：$E_1(t) = T_m(t) - 100 = \boxed{-80e^{-t/10}}$

在 $t = 60$ s 时传感器温度：
$$T_m(60) = 100 - 80e^{-60/10} = 100 - 80e^{-6} \approx 99.80°C$$

**阶段 2 — 回到空气中 (t > 60 s)**：
- 定义新时间变量 $t' = t - 60$
- $$T_m(t) = 20 + (99.80 - 20)e^{-(t-60)/50} = 20 + 79.80e^{-(t-60)/50}$$
- **动态误差**：$E_2(t) = T_m(t) - 20 = \boxed{79.80e^{-(t-60)/50}}$

#### Step 3: 计算各时刻误差

| $t$ (s) | 所属阶段 | 公式 | 动态误差 $E(t)$ |
|---------|---------|------|----------------|
| 10 | 阶段 1 | $-80e^{-10/10}$ | $-80 \times 0.3679 = \boxed{-29.43°C}$ |
| 20 | 阶段 1 | $-80e^{-20/10}$ | $-80 \times 0.1353 = \boxed{-10.83°C}$ |
| 50 | 阶段 1 | $-80e^{-50/10}$ | $-80 \times 0.00674 = \boxed{-0.54°C}$ |
| 120 | 阶段 2 | $79.80e^{-60/50}$ | $79.80 \times 0.3012 = \boxed{+24.04°C}$ |
| 300 | 阶段 2 | $79.80e^{-240/50}$ | $79.80 \times 0.00823 = \boxed{+0.66°C}$ |

> 📊 **物理含义**: 负误差 = 传感器还没跟上（落后于真实温度）; 正误差 = 传感器还没冷却下来（残留前一阶段的热量）。

### 🎯 做题技巧

1. **$\tau = \frac{mC}{hA}$ 是热系统的核心公式** — 记住它，考试直接套。
2. **"两次阶跃"问题分阶段处理**：每个阶段用各自的时间常数和初始条件。第一阶段结束时温度是第二阶段的初始条件。
3. **动态误差定义**：$E(t) = T_{measured} - T_{true}$，注意正负号含义。
4. **$e^{-5} \approx 0.0067$，$e^{-6} \approx 0.0025$** — 记住这几个关键值可以快速估算。

## 习题 2: 二阶力传感器 — 阶跃响应

### 题目

> 一个力传感器：质量 $m = 0.5$ kg，刚度 $k = 2 \times 10^2$ N/m，阻尼常数 $\lambda = 6.0$ N·s/m。
>
> (a) 计算稳态灵敏度、自然频率和阻尼比。
> (b) 计算 2 N 稳态力作用下的位移。
> (c) 输入力从 2 N 突增到 3 N，推导位移表达式。

### 解答

#### (a) 三个核心参数

$$\boxed{K = \frac{1}{k} = \frac{1}{200} = 0.005 \text{ m/N}}$$

$$\boxed{\omega_n = \sqrt{\frac{k}{m}} = \sqrt{\frac{200}{0.5}} = \sqrt{400} = 20 \text{ rad/s}}$$

$$\boxed{\xi = \frac{\lambda}{2\sqrt{km}} = \frac{6.0}{2\sqrt{200 \times 0.5}} = \frac{6.0}{2\sqrt{100}} = \frac{6.0}{20} = 0.3}$$

#### (b) 稳态位移

稳态时无加速度和速度，阻尼和惯性力为零：

$$x(0) = K \cdot F = 0.005 \times 2 = \boxed{0.01 \text{ m}}$$

#### (c) 阶跃响应表达式

$\Delta F = 3 - 2 = 1$ N，$\Delta x_{steady} = K \times \Delta F = 0.005 \times 1 = 0.005$ m。

**计算中间参数**：
- 衰减系数：$\xi\omega_n = 0.3 \times 20 = 6$
- 阻尼自然频率：$\omega_d = \omega_n\sqrt{1-\xi^2} = 20\sqrt{1-0.09} = 20\sqrt{0.91} \approx 19.08$ rad/s
- 正弦项系数：$\frac{\xi}{\sqrt{1-\xi^2}} = \frac{0.3}{\sqrt{0.91}} \approx 0.314$

**欠阻尼阶跃响应标准公式**：

$$\boxed{\Delta x(t) = \Delta x_{steady}\left[1 - e^{-\xi\omega_n t}\left(\cos\omega_d t + \frac{\xi}{\sqrt{1-\xi^2}}\sin\omega_d t\right)\right]}$$

代入数值：

$$\Delta x(t) = 0.005\left[1 - e^{-6t}(\cos 19.08t + 0.314\sin 19.08t)\right]$$

**总位移**：

$$\boxed{x(t) = x(0) + \Delta x(t) = 0.01 + 0.005\left[1 - e^{-6t}(\cos 19.08t + 0.314\sin 19.08t)\right] \text{ m}}$$

### 🎯 做题技巧

1. **力传感器的稳态灵敏度 $K = 1/k$**（不是 $k$！）— 力学传感器的输入是力，输出是位移。
2. **$\omega_n = \sqrt{k/m}$，$\xi = \lambda/(2\sqrt{km})$** — 这两个公式闭眼默写。
3. **阶跃响应用"位移增量"叠加**：先算初始稳态位移，再加阶跃变化量 $\Delta x(t)$。
4. **欠阻尼标准公式**直接套，中间参数 $\omega_d$ 和 $\xi/\sqrt{1-\xi^2}$ 先算好。

## 习题 3: 二阶系统 — 周期性输入与动态误差

### 题目

> 一个力测量系统整体灵敏度为 1，动态特性由二阶传感元件决定：$\omega_n = 40$ rad/s，$\xi = 0.1$。计算输入力信号的系统动态误差：
> $$F(t) = 50\left(\sin 10t + \frac{1}{3}\sin 30t + \frac{1}{5}\sin 50t\right)$$

### 解答

#### Step 1: 分解频率分量

| 分量 | 频率 $\omega$ | 振幅 $A$ |
|------|-------------|----------|
| 基频 (Fundamental) | $\omega_1 = 10$ rad/s | $A_1 = 50$ |
| 三次谐波 (3rd Harmonic) | $\omega_2 = 30$ rad/s | $A_2 = 50/3 \approx 16.67$ |
| 五次谐波 (5th Harmonic) | $\omega_3 = 50$ rad/s | $A_3 = 50/5 = 10$ |

#### Step 2: 对每个频率计算幅值比和相移

**二阶系统幅值比和相移公式**：

$$|G(j\omega)| = \frac{1}{\sqrt{\left(1 - \frac{\omega^2}{\omega_n^2}\right)^2 + \left(2\xi\frac{\omega}{\omega_n}\right)^2}}$$

$$\phi = -\arctan\left(\frac{2\xi\frac{\omega}{\omega_n}}{1 - \frac{\omega^2}{\omega_n^2}}\right)$$

**频率 1: $\omega_1 = 10$ rad/s**

频率比 = $10/40 = 0.25$

$$|G_1| = \frac{1}{\sqrt{(1-0.0625)^2 + (0.05)^2}} = \frac{1}{\sqrt{0.9375^2 + 0.05^2}} \approx 1.065$$

$$\phi_1 = -\arctan\left(\frac{0.05}{0.9375}\right) \approx -3.05°$$

$$O_1(t) = 50 \times 1.065 \sin(10t - 3.05°) = \boxed{53.25\sin(10t - 3.05°)}$$

**频率 2: $\omega_2 = 30$ rad/s**

频率比 = $30/40 = 0.75$

$$|G_2| = \frac{1}{\sqrt{(1-0.5625)^2 + (0.15)^2}} = \frac{1}{\sqrt{0.4375^2 + 0.15^2}} \approx 2.162$$

$$\phi_2 = -\arctan\left(\frac{0.15}{0.4375}\right) \approx -18.92°$$

$$O_2(t) = 16.67 \times 2.162 \sin(30t - 18.92°) = \boxed{36.04\sin(30t - 18.92°)}$$

**频率 3: $\omega_3 = 50$ rad/s**

频率比 = $50/40 = 1.25$

$$|G_3| = \frac{1}{\sqrt{(1-1.5625)^2 + (0.25)^2}} = \frac{1}{\sqrt{(-0.5625)^2 + 0.25^2}} \approx 1.625$$

> ⚠️ **易错点！** 分母实部 $1 - (\omega/\omega_n)^2 = -0.5625 < 0$，相移已超过 −90° 进入第三象限，必须**减去 180°**：

$$\phi_3 = -\arctan\left(\frac{0.25}{-0.5625}\right) - 180° = 23.96° - 180° = -156.04°$$

$$O_3(t) = 10 \times 1.625 \sin(50t - 156.04°) = \boxed{16.25\sin(50t - 156.04°)}$$

#### Step 3: 计算动态误差

$$\boxed{E(t) = [O_1(t) + O_2(t) + O_3(t)] - F(t)}$$

$$E(t) = [53.25\sin(10t-3.05°) - 50\sin 10t]$$
$$+ [36.04\sin(30t-18.92°) - 16.67\sin 30t]$$
$$+ [16.25\sin(50t-156.04°) - 10\sin 50t]$$

### 🎯 做题技巧

1. **叠加原理是关键**：总输出 = 各频率分量独立响应的和。
2. **二阶系统幅值比公式**分母有两个关键项：$(1 - (\omega/\omega_n)^2)^2$ (刚度/惯性) + $(2\xi\omega/\omega_n)^2$ (阻尼)。
3. **相移象限判断**至关重要：当 $\omega > \omega_n$ 时分母实部变负，要 **-180°**。
4. **共振区放大**：$\omega \approx \omega_n$ 且 $\xi$ 小时幅值比远大于 1（此处 $\omega_3/\omega_n = 1.25$，$|G| = 1.625$）。

## 习题 4: 完整传感系统 — 压电晶体 + 电荷放大器 + 记录仪

### 题目

> 系统组成：压电晶体 → 电荷放大器（$G(s) = \frac{10^{-3} \tau s}{1 + \tau s}$，$\tau = 0.1$ s）→ 记录仪（$G(s) = \frac{50}{\frac{s^2}{\omega_n^2} + 2\xi s/\omega_n + 1}$，$\omega_n = 50$ rad/s，$\xi = 0.2$）
>
> (a) 对同样的输入 $F(t) = 50(\sin 10t + \frac{1}{3}\sin 30t + \frac{1}{5}\sin 50t)$ 计算动态误差。
> (b) 简要说明减少误差的系统修改方案。

### 解答

#### (a) 计算系统动态误差

**Step 1: 推导总传递函数**

$$G(s) = 20 \cdot \frac{10^{-3} \times 0.1s}{1 + 0.1s} \cdot \frac{50}{(s/50)^2 + 2(0.2)(s/50) + 1}$$

常数乘积 $20 \times 10^{-3} \times 50 = 1$（静态增益完全抵消）。

频域形式：

$$G(j\omega) = \underbrace{\left(\frac{j0.1\omega}{1 + j0.1\omega}\right)}_{\text{高通 } H_A} \cdot \underbrace{\left(\frac{1}{[1 - (\omega/50)^2] + j0.008\omega}\right)}_{\text{低通 } H_B}$$

> ✅ **关键洞见**：电荷放大器是一阶**高通**滤波器（衰减低频），记录仪是二阶**低通**滤波器（衰减高频）。系统是**带通**特性。

**Step 2: 三个频率分量计算**

**频率 1: $\omega_1 = 10$ rad/s**

| 环节 | 计算 |
|------|------|
| 高通 $H_A$ | $\frac{j}{1+j}$，$\vert H_A\vert = 1/\sqrt{2} \approx 0.707$，$\angle H_A = 90° - 45° = 45°$ |
| 低通 $H_B$ | $\frac{1}{0.96 + j0.08}$，$\vert H_B\vert \approx 1.038$，$\angle H_B \approx -4.76°$ |
| **总** | $\vert G\vert = 0.707 \times 1.038 = 0.734$，$\angle G = 45° - 4.76° = 40.24°$ |

$$\boxed{O_1(t) = 36.70\sin(10t + 40.24°)}$$

**频率 2: $\omega_2 = 30$ rad/s**

| 环节 | 计算 |
|------|------|
| 高通 $H_A$ | $\frac{j3}{1+j3}$，$\vert H_A\vert = 3/\sqrt{10} \approx 0.949$，$\angle H_A = 90° - \arctan(3) \approx 18.43°$ |
| 低通 $H_B$ | $\frac{1}{0.64 + j0.24}$，$\vert H_B\vert \approx 1.463$，$\angle H_B \approx -20.56°$ |
| **总** | $\vert G\vert = 0.949 \times 1.463 \approx 1.388$，$\angle G \approx -2.13°$ |

$$\boxed{O_2(t) = 23.14\sin(30t - 2.13°)}$$

**频率 3: $\omega_3 = 50$ rad/s（共振点！）**

| 环节 | 计算 |
|------|------|
| 高通 $H_A$ | $\frac{j5}{1+j5}$，$\vert H_A\vert = 5/\sqrt{26} \approx 0.981$，$\angle H_A \approx 11.31°$ |
| 低通 $H_B$ | 分母 $= j0.4$（实部 = 0，共振），$\vert H_B\vert = 1/0.4 = 2.5$，$\angle H_B = -90°$ |
| **总** | $\vert G\vert = 0.981 \times 2.5 \approx 2.453$，$\angle G \approx -78.69°$ |

$$\boxed{O_3(t) = 24.53\sin(50t - 78.69°)}$$

**Step 3: 动态误差**

$$E(t) = [36.70\sin(10t+40.24°) - 50\sin 10t]$$
$$+ [23.14\sin(30t-2.13°) - 16.67\sin 30t]$$
$$+ [24.53\sin(50t-78.69°) - 10\sin 50t]$$

> 📊 基频被**衰减**（0.734），五次谐波被**严重放大**（2.453 × 共振）。系统性能很差。

#### (b) 系统修改方案

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| **低频衰减**（基频 $|G| = 0.734$） | 电荷放大器的高通截止频率 $\omega_c = 1/\tau = 10$ rad/s **太高**，侵入工作频带 | **增大时间常数 $\tau$**（增大反馈电容/电阻），使高通截止频率远低于 10 rad/s |
| **高频共振**（五次谐波 $|G| = 2.453$） | 记录仪的 $\omega_n = 50$ rad/s 正好与五次谐波重合，且 $\xi = 0.2$ 过低 | **增加阻尼**至 $\xi \approx 0.7$（最优值）；**提高 $\omega_n$** 使其远高于最高信号频率 |

### 🎯 做题技巧

1. **识别滤波器类型**：$s/(1+\tau s)$ 是高通，$1/(s^2+2\xi\omega_n s+\omega_n^2)$ 是低通。级联 = 带通。
2. **先检查静态增益是否抵消**（此处 $20 \times 10^{-3} \times 50 = 1$），简化计算。
3. **共振点判断**：当 $\omega = \omega_n$ 时分母实部为 0，$|H_B| = 1/(2\xi) = 1/0.4 = 2.5$。
4. **工程诊断思路**：低频衰减 → 高通截止频率太高 → 增大 $\tau$；高频共振 → 阻尼太小或 $\omega_n$ 太低 → 增 $\xi$ 或增 $\omega_n$。

## 习题 5: 热电偶开环动态补偿

### 题目

> 一个未补偿的热电偶在快速流动液体中的时间常数为 10 s。
>
> (a) 计算热电偶频率响应的带宽。
> (b) 找出未补偿热电偶幅值比在 5% 误差范围内的频率范围。
> (c) 采用超前/滞后电路 $G_c(s) = \frac{1+10s}{1+s}$ 补偿后，找出幅值比在 5% 误差内的频率范围。
> (d) 液体流速降低导致热电偶时间常数增加到 20 s。通过绘制 $|G(j\omega)|$ 解释为何补偿效果恶化。

### 解答

#### (a) 带宽

一阶系统带宽定义为幅值比降至 $1/\sqrt{2} \approx 0.707$（−3 dB 截止频率）处：

$$\boxed{\omega_c = \frac{1}{\tau} = \frac{1}{10} = 0.1 \text{ rad/s}}$$

#### (b) 未补偿系统的 5% 误差带

5% 误差 → 幅值比 $|G_u(j\omega)| \geq 0.95$：

$$\frac{1}{\sqrt{1 + (10\omega)^2}} \geq 0.95$$

$$1 + 100\omega^2 \leq \left(\frac{1}{0.95}\right)^2 \approx 1.108$$

$$100\omega^2 \leq 0.108 \quad \Rightarrow \quad \omega^2 \leq 0.00108$$

$$\boxed{\omega \leq 0.0329 \text{ rad/s}}$$

> 📊 只有非常低的频率（周期 > 3 分钟）才能被准确测量！

#### (c) 补偿后的 5% 误差带

补偿后的总传递函数（零极点对消）：

$$G_{comp}(s) = G_u(s) \cdot G_c(s) = \frac{1}{1+10s} \cdot \frac{1+10s}{1+s} = \boxed{\frac{1}{1 + s}}$$

系统变为 $\tau_{new} = 1$ s 的新一阶系统：

$$\frac{1}{\sqrt{1 + \omega^2}} \geq 0.95$$

$$\omega^2 \leq 0.108 \quad \Rightarrow \quad \boxed{\omega \leq 0.329 \text{ rad/s}}$$

> ✅ **提升 10 倍！** 有用带宽从 0.033 rad/s 提升到 0.329 rad/s。

#### (d) 参数失配导致的补偿失效

当 $\tau$ 从 10 s → 20 s（流速降低，热惯性增大）：

| 因素 | 说明 |
|------|------|
| **根本原因** | "零极点失配"：补偿电路的零点（$\tau = 10$ s）无法消除传感器实际慢极点（$\tau = 20$ s） |
| **中频凹陷** | 实际传感器在 $\omega = 1/20 = 0.05$ rad/s 提前开始衰减，而补偿器零点在 $\omega = 1/10 = 0.1$ rad/s 才开始"拉升" → **时序错配**产生巨大凹陷 |
| **性能恶化** | 中频段（0.05 ~ 0.3 rad/s）幅值比可能**低于未补偿基线** |
| **工程教训** | 开环动态补偿**极其脆弱** — 依赖完美的数学对消，对环境变化毫无鲁棒性 |

### 🎯 做题技巧

1. **一阶系统带宽**：$\omega_c = 1/\tau$，直接记住。
2. **5% 误差 → $|G| \geq 0.95$**，解不等式 $1/\sqrt{1+(\tau\omega)^2} \geq 0.95$。
3. **零极点对消公式**：$\frac{1}{1+\tau_{slow}s} \cdot \frac{1+\tau_{slow}s}{1+\tau_{fast}s} = \frac{1}{1+\tau_{fast}s}$。
4. **补偿失效的根因**：参数漂移 → 完美对消失败 → 开环补偿的致命弱点。考试常问"为什么补偿会失效"。

## 习题 6: 弹性力传感器与闭环力平衡加速度计

### 题目

> 弹性力传感器：有效质量 $m = 0.1$ kg，弹簧刚度 $k = 10$ N/m，阻尼常数 $\lambda = 14$ N·s/m。
>
> (a) 计算：(i) 自然频率 (ii) 阻尼比 (iii) 位移-力传递函数
>
> (b) 将传感器纳入**闭环力平衡加速度计**，额外组件：电位计位移传感器（灵敏度 1.0 V/m）、放大器（电压输入电流输出，40 A/V）、线圈和磁铁（电流输入力输出，25 N/A）、电阻 250 Ω。
>
> (i) 绘制加速度计框图。
> (ii) 计算整体传递函数。
> (iii) 解释为什么加速度计的动态性能优于弹性传感器。

### 解答

#### (a) 三个核心参数

**(i) 自然频率**：
$$\boxed{\omega_n = \sqrt{\frac{k}{m}} = \sqrt{\frac{10}{0.1}} = \sqrt{100} = 10 \text{ rad/s}}$$

**(ii) 阻尼比**：
$$\boxed{\xi = \frac{\lambda}{2\sqrt{km}} = \frac{14}{2\sqrt{10 \times 0.1}} = \frac{14}{2 \times 1} = 7.0}$$

> ⚠️ $\xi = 7.0 \gg 1$：这是**严重过阻尼**系统！响应极慢，远非理想。

**(iii) 传递函数**：
$$\boxed{G_s(s) = \frac{x(s)}{F(s)} = \frac{1/k}{\frac{s^2}{\omega_n^2} + \frac{2\xi s}{\omega_n} + 1} = \frac{0.1}{0.01s^2 + 1.4s + 1}}$$

#### (b) 闭环系统

**(i) 框图**：

```
a_in → [Σ] → [弹性传感器 G_s(s)] → x → [电位计 1 V/m] → V → [放大器 40 A/V] → I → [线圈 25 N/A] → F_fb → [−]
       ↑                                                                                                    |
       └──────────────────────────────── [电阻 250 Ω] ←────────────────────────────────────────────────────┘
```

**(ii) 传递函数**：

前向增益 $K_F = 40 \times 25 = 1000$ N/V（放大器的 A/V × 线圈的 N/A）。

反馈路径 $K_B = 1.0$ V/m（电位计将位移转回电压）。

闭环传递函数标准形式：

$$\frac{x}{a}(s) = \frac{m \cdot G_s(s)}{1 + K_F K_B \cdot G_s(s)}$$

代入后得到远优于开环的动态特性：闭环自然频率 $\omega_{n,c} \gg \omega_n$，闭环阻尼比 $\xi_c$ 可调至最优值 0.7。

**(iii) 闭环优越性**：

| 指标 | 开环弹性传感器 | 闭环力平衡加速度计 |
|------|-------------|-----------------|
| $\omega_n$ | 10 rad/s（低） | 远高于 10 rad/s（由反馈增益提升） |
| $\xi$ | 7.0（严重过阻尼，响应极慢） | 可调至 **0.7**（最优） |
| 带宽 | 极窄 | 大幅扩展 |
| 机械参数依赖性 | $K = 1/k$，$k$ 会老化和温漂 | $K$ 取决于电子元件（电阻、磁场），高度稳定 |
| 鲁棒性 | 差 | 强 |

### 🎯 做题技巧

1. **弹性传感器**：$\omega_n = \sqrt{k/m}$，$K = 1/k$（位移/力），$\xi = \lambda/(2\sqrt{km})$。
2. **闭环框图**：传感器 → 位移传感器 → 放大器 → 执行器（线圈）→ 力反馈至输入端，构成负反馈。
3. **闭环优势三点**：更高 $\omega_{n,c}$、最优 $\xi_c$、精度不再依赖老化机械参数。
4. **$\xi = 7.0$ 是考题的"钓鱼"数值**：正常 $\xi$ 应在 0.1–1 范围，7.0 意味着严重过阻尼，响应极慢。闭环反馈是唯一的"解药"。

## 综合技巧总结

| 题型 | 解题套路 | 易错点 |
|------|---------|--------|
| **一阶阶跃** | $\tau = mC/(hA)$ → $T(t) = T_\infty + (T_0 - T_\infty)e^{-t/\tau}$ → $E(t) = T_m - T_{true}$ | 两次阶跃的初始条件传递 |
| **二阶参数** | $K = 1/k$，$\omega_n = \sqrt{k/m}$，$\xi = \lambda/(2\sqrt{km})$ | 力传感器 $K = 1/k$，不是 $k$ |
| **周期输入** | 分解频率 → 每个频率算 $|G|$ 和 $\phi$ → 叠加 → $E(t) = O(t) - I(t)$ | $\omega > \omega_n$ 时相移要 **-180°** |
| **级联系统** | 先检查静态增益抵消 → 频域每个 $\omega$ 级联计算 | 识别高通/低通级联 = 带通 |
| **滤波/补偿** | 高通截止 $\omega_c = 1/\tau$，零极点对消 $G_{comp} = G_s \cdot G_c$ | 参数漂移 → 对消失败 |
| **闭环反馈** | $G_{cl} = G_F/(1 + G_F G_B)$，$\omega_{n,c} \gg \omega_n$，$\xi_c$ 可调 | 框图方向不要画反 |
