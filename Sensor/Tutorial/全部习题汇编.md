# Tutorial 1 — 静态特性 (Static Characteristics)

---

## T1.1 理想直线方程

**题目**：力传感器输入 0~$2\times10^5$ N → 输出 1~5 V。差压变送器输入 0~$2\times10^4$ Pa → 输出 4~20 mA。分别写出理想直线方程 $O=KI+a$。

**解**：

力传感器：
$$K = \frac{5-1}{2\times10^5-0} = 2\times10^{-5}\ \mathrm{V/N},\quad a = 1\ \mathrm{V}$$
$$\boxed{V_{ideal} = 2\times10^{-5}\cdot F + 1}$$

差压变送器：
$$K = \frac{20-4}{2\times10^4-0} = 8\times10^{-4}\ \mathrm{mA/Pa},\quad a = 4\ \mathrm{mA}$$
$$\boxed{i_{ideal} = 8\times10^{-4}\cdot \Delta P + 4}$$


---

## T1.2 非线性 — 压力传感器

**题目**：压力传感器 0~10 bar → 输出 0~5 V（理想直线）。在 4 bar 时实际输出 2.20 V。求该点的非线性（伏特值和 %f.s.d.）。

**解**：
$$K = 5/10 = 0.5\ \mathrm{V/bar},\quad O_{ideal}(4) = 0.5\times4 = 2.0\ \mathrm{V}$$
$$N(4) = 2.20 - 2.0 = \boxed{+0.20\ \mathrm{V}}$$
$$\%\mathrm{f.s.d.} = \frac{0.20}{5-0}\times100\% = \boxed{4\%}$$


---

## T1.3 非线性 — 温度传感器

**题目**：温度传感器 0~400°C → 输出 0~20 mV（线性）。在 100°C 时实际输出 4.5 mV。求非线性。

**解**：
$$K = 20/400 = 0.05\ \mathrm{mV/{}^\circ C},\quad O_{ideal}(100) = 5.0\ \mathrm{mV}$$
$$N(100) = 4.5 - 5.0 = \boxed{-0.5\ \mathrm{mV}}$$
$$\%\mathrm{f.s.d.} = \frac{0.5}{20}\times100\% = \boxed{2.5\%}$$

## T1.4 非线性 — 热电偶数据表

**题目**：热电偶 5 点标定数据（500°C 时 27388 μV）。用端点法求 $K$，逐点算非线性。

**解**：

| 200 | 10777 | 10955.2 | −178.2 | 0.65% |
| 300 | 16325 | 16432.8 | −107.8 | 0.39% |

## T1.5 环境效应 — 纯修正输入 ($K_M$)

**题目**：力传感器 20°C 时 0~10 kN → 0~5 V，30°C 时 0~10 kN → 0~5.5 V。求 $K_M$。

**解**：
$$K(20) = 0.5\ \mathrm{V/kN},\quad K(30) = 0.55\ \mathrm{V/kN}$$
零点均为 0 V → 灵敏度变了，零点没变 → 纯 $K_M$，$K_I=0$。
$$\boxed{K_M = \frac{0.55-0.5}{30-20} = 0.005\ \mathrm{V/(kN\cdot{}^\circ C)}}$$


---

## T1.6 环境效应 — 纯干扰输入 ($K_I$)

**题目**：传感器 20°C 时 1.0~5.0 V，30°C 时 1.2~5.2 V。求 $K_I$。

**解**：
量程均为 4.0 V → 斜率不变。零点从 1.0→1.2 V。
$$\boxed{K_I = \frac{1.2-1.0}{10} = 0.02\ \mathrm{V/{}^\circ C}}$$

## T1.7 环境效应 — 混合型 ($K_M+K_I$)

**题目**：变送器 20°C 时 0~10⁴ Pa → 4~20 mA，30°C 时 0~10⁴ Pa → 4.2~20.8 mA。求 $K_I$ 和 $K_M$。

**解**：
20°C: $K=1.6\times10^{-3}$ mA/Pa, 零点=4.0 mA
30°C: $K=1.66\times10^{-3}$ mA/Pa, 零点=4.2 mA
$$\boxed{K_I = \frac{4.2-4.0}{10} = 0.02\ \mathrm{mA/{}^\circ C}}$$
$$\boxed{K_M = \frac{1.66\times10^{-3}-1.6\times10^{-3}}{10} = 6\times10^{-6}\ \mathrm{mA/(Pa\cdot{}^\circ C)}}$$

## T1.8 ADC 分辨率

**题目**：8-bit 和 16-bit ADC，输入 0~5 V。求电压分辨率和 %f.s.d.。

**解**：
| 位数 | $2^n$ | $\Delta V$ | %f.s.d. |
|------|-------|-----------|---------|
| 8-bit | 256 | $5/256 = 19.53$ mV | $1/256 = 0.391\%$ |
| 16-bit | 65536 | $5/65536 = 76.3$ μV | $1/65536 = 0.00153\%$ |

## T1.9 迟滞 (Hysteresis)

**题目**：液位传感器 3 m 处上升输出 2.95 V，下降输出 3.05 V。满量程 0~10 V。求迟滞。

**解**：
$$H = 2.95 - 3.05 = \boxed{-0.10\ \mathrm{V}}$$
$$\%\mathrm{span} = \frac{0.10}{10}\times100\% = \boxed{1\%}$$

## T2.1 串联系统误差

**题目**：力测量系统 $K_1=10^{-2}$, $K_2=5\times10^{-2}$, $K_3=10^3$, $K_4=1.9$，真实力 10 kN。求系统误差。液位系统 $K_1=0.050$, $K_2=21.5$, $K_3=0.99$，液位 5 m。

**解**：

力系统：
$$K_{sys} = 0.01\times0.05\times1000\times1.9 = 0.95$$
$$\boxed{E = (0.95-1)\times10 = -0.5\ \mathrm{kN}}$$

液位系统：
$$K_{sys} = 0.050\times21.5\times0.99 = 1.06425$$
$$\boxed{E = (1.06425-1)\times5.0 = +0.321\ \mathrm{m}}$$


---

## T2.2 误差带级联传播

**题目**：4 级级联系统（传感器 $K=10^{-4}$, $h=0.005$ → 电桥 → 放大器 $K=10^3$, $h=0.5$ → 记录仪 $K=250$, $h=100$）。真实输入 5000 Pa。求 (a) $\sigma_E$，(b) 记录仪灵敏度误调至 225 后的系统偏差。

**解**：

(a) 逐级递推（$\sigma_{out}^2 = (K\sigma_{in})^2 + h^2/3$）：

第 1 级：$\sigma_1^2 = 0 + 0.005^2/3 \approx 8.33\times10^{-6}$

第 2 级：$\sigma_2^2 = (4\times10^{-2})^2\times8.33\times10^{-6} + (5\times10^{-4})^2/3 \approx 9.66\times10^{-8}$

第 3 级：$\sigma_3^2 = 10^6\times9.66\times10^{-8} + 0.5^2/3 \approx 0.18$

第 4 级：$\sigma_4^2 = 250^2\times0.18 + 100^2/3 \approx 14583.3$

$$\boxed{\sigma_E = \sqrt{14583.3} \approx 120.8\ \mathrm{Pa}}$$

(b) $K_{actual} = 10^{-4}\times4\times10^{-2}\times10^3\times225 = 0.9$
$$\boxed{\bar{E} = 0.9\times5000 - 5000 = -500\ \mathrm{Pa}}$$

供压变化 50%，输出仅变 0.33% — 抑制比约 150。

## T2.5 双轨统计 — 热电偶系统

**题目**：热电偶 $E = C_0 + C_1 T + C_2 T^2$ → 转换器 $i = K_1 E + K_v E\Delta T_a + K_2\Delta T_a + a_1$ → 记录仪 $T_M = K_2 i + a_2$。$T=117$°C。各参数有均值和标准差（正态分布）。求 $\bar{E}$ 和 $\sigma_E$。

**解**：

**均值轨**：代入标称值。

$E = 0.04017\times117 + 4.66\times10^{-6}\times117^2 = 4.764$ mV

$i = 3.893\times4.764 + 1.95\times10^{-4}\times4.764\times(-10) + 2.0\times10^{-3}\times(-10) - 3.864 = 14.652$ mA

$T_M = 6.25\times14.652 + 25.0 = 116.57$°C

$$\boxed{\bar{E} = 116.57 - 117 = -0.43\ \mathrm{{}^\circ C}}$$

**方差轨**：逐级传播偏导$\times$方差。
$$\boxed{\sigma_E \approx 1.93\ \mathrm{{}^\circ C}}$$

## T2.6 皮托管流速系统 — 诊断与优化

**题目**：皮托管 $\Delta P = 0.5\rho v^2$ → 变送器 $i = K_1\Delta P + a_1$ → ADC $n = K_2 i$ → 微控制器 $v_M = 1.430\sqrt{n-51}$。$v_T=14$ m/s，$\rho=1.2$。求 $\bar{E}$ 和 $\sigma_E$，诊断改进。

**解**：

均值得：

| 元件 | 输出 |
|------|------|
| 皮托管 | $\Delta P = 0.5\times1.2\times14^2 = 117.6$ Pa |
| 变送器 | $i = 0.064\times117.6 + 4.0 = 11.5264$ mA |
| ADC | $n = 12.80\times11.5264 = 147.54$ |
| 微控制器 | $v_M = 1.430\sqrt{147.54-51} = 14.050$ m/s |

$$\boxed{\bar{E} = 14.050 - 14.0 = +0.050\ \mathrm{m/s}}$$

$$\boxed{\sigma_E \approx 0.34\ \mathrm{m/s}}$$

| 问题 | 方案 |
|------|------|
| 系统偏差 +0.05 | 改软件：用精确逆公式 $v=1.4264\sqrt{n-51.2}$ |
| 随机波动 0.34 | 加硬件：温压传感器实时算 $\rho=P/(RT)$ |

## T2.7 热敏电阻双轨统计

**题目**：热敏电阻 $R_\theta = K_1 e^{B/\theta}$ → 电桥 $V_o = f(R_\theta)$ → 记录仪 $T_M = 50V_o+300$。$\theta=320$ K，各参数带均值和标准差。求 $\bar{E}$ 和 $\sigma_E$。

$T_M = 50\times0.3867 + 300 = 319.34$ K
$$\boxed{\bar{E} = 319.34 - 320 = -0.67\ \mathrm{K}}$$

方差轨：
$$\boxed{\sigma_E \approx 4.82\ \mathrm{K}}$$

## T3.2 二阶力传感器 — 阶跃响应

**题目**：$m=0.5$ kg, $k=200$ N/m, $\lambda=6.0$ Ns/m, 力从 2→3 N 阶跃。求 $K$, $\omega_n$, $\xi$，写位移表达式。

**解**：
$$K = 1/200 = \boxed{0.005\ \mathrm{m/N}}$$
$$\omega_n = \sqrt{200/0.5} = \boxed{20\ \mathrm{rad/s}}$$
$$\xi = 6.0/(2\sqrt{100}) = \boxed{0.3}$$

$$\boxed{x(t) = 0.01 + 0.005[1 - e^{-6t}(\cos 19.08t + 0.314\sin 19.08t)]\ \mathrm{m}}$$

## T3.4 级联系统 — 压电+电荷放大+记录仪

## T4.1 热电偶加载效应

**题目**：热电偶灵敏度 0.04 mV/°C，内阻 100 Ω，接 1 kΩ 负载，测 250°C。求负载两端电压。

**解**：
$$E_{Th} = 0.04\times250 = 10\ \mathrm{mV}$$
$$V_L = 10\times\frac{1000}{100+1000} = \boxed{9.09\ \mathrm{mV}}$$

加载损失：0.91 mV（9.1%）。

## T4.2 电位器加载 — $E_{Th}$ vs $V_L$

**题目**：电位器 $V_S=10$ V, $R_P=10$ kΩ, 全长 10 cm。接记录仪 $R_L=10$ kΩ。求 $x=0.2,0.5,0.8$ 时的 $E_{Th}$, $R_{Th}$, $V_L$。

**解**：

$$E_{Th} = V_S \cdot x,\quad R_{Th} = R_P\cdot x(1-x),\quad V_L = E_{Th}\cdot\frac{R_L}{R_{Th}+R_L}$$

| $x$ | $E_{Th}$ | $R_{Th}$ | $V_L$ | 损失 |
|-----|----------|----------|-------|------|
| 0.2 | 2.0 V | 1.6 kΩ | 1.72 V | 14% |
| 0.5 | 5.0 V | 2.5 kΩ | 4.0 V | 20% |
| 0.8 | 8.0 V | 1.6 kΩ | 6.90 V | 14% |

中心 ($x=0.5$) 损失最大（$R_{Th}$ 抛物线顶点）。

## T4.4 自相关函数 (ACF)

## T4.5 噪声 PSD / SNR / 滤波 / 平均

**题目**：信号 1.4 mV 峰值 5 kHz 正弦，白噪声 PSD 100 pW/Hz，带宽 1 MHz。求 (a) $W_N$, $V_{N,rms}$ (b) 初始 SNR (c) ACF 草图 (d) 1 kHz 带通滤波后 SNR (e) 100 次平均后 SNR。

**解**：

(a) $W_N = 10^{-10}\times10^6 = \boxed{100\ \mathrm{\mu W}}$, $V_{N,rms} = \sqrt{10^{-4}} = \boxed{10\ \mathrm{mV}}$

(b) $V_{s,rms}=1.4/\sqrt{2}\approx1$ mV, $W_S = 1$ μW。$SNR = 10\log_{10}(1/100) = \boxed{-20\ \mathrm{dB}}$

(c) 总 ACF = 信号 ACF（余弦振荡 $W_S\cos(2\pi f_s\beta)$）+ 噪声 ACF（原点 $\delta$ 尖峰 $W_N$）

(d) 带宽 1000× 缩小：$W_N' = 0.1$ μW。$SNR' = 10\log_{10}(10) = \boxed{10\ \mathrm{dB}}$。改善 $30$ dB。

(e) $10\log_{10}(100) = 20$ dB 进一步改善 → 最终 $\boxed{30\ \mathrm{dB}}$。

总改善 50 dB（100,000×）。

## T4.6 接地回路干扰 — 串模 vs 共模

**题目**：热电偶 10 mV DC，DVM CMRR=100 dB，两地间 100V 50Hz 干扰，1000 pF 泄漏电容。求串模、共模干扰及电压表读数范围。

**解**：

$X_C = 1/(2\pi\times50\times10^{-9}) \approx 3.18$ MΩ。$i_E = 100/3.18\times10^6 \approx 31.4$ μA。

$$V_{SM} = 31.4\ \mathrm{\mu A}\times100\ \Omega = \boxed{3.14\ \mathrm{mV\ (rms)}}$$
$$V_{CM} \approx \boxed{100\ \mathrm{V\ (rms)}}$$

## T10.6 圆柱力传感器 — 泊松差分全桥

**题目**：钢柱 $A=2\times10^{-3}$ m², $F=200$ kN, $E=2\times10^{11}$ Pa, $\nu=0.3$。两纵两横四片 $G=2.0$, $R=120$ Ω, $U_i=10$ V。求全桥输出和放大器增益（目标 5 V）。

# 全局技巧速查

## 公式分类索引

**静态特性**：$K=\Delta O/\Delta I$, $a=O_{min}-KI_{min}$, $N(I)=O(I)-O_{ideal}(I)$, $\%=\|N\|/span$

**环境效应**：$K_M=\Delta K/\Delta I_M$, $K_I=\Delta a/\Delta I_I$

**ADC/DAC**：$\Delta V=V_{FS}/2^n$（ADC），$\Delta V=V_{FS}/(2^n-1)$（DAC）

**串联误差**：$K_{sys}=\prod K_i$, $E=(K_{sys}-1)I$

**级联方差**：$\sigma_{out}^2=(K\sigma_{in})^2+h^2/3$（均匀分布 $\sigma^2=h^2/3$）

**双轨统计**：均值轨 = 逐级代入标称值；方差轨 = $\sum(\partial f/\partial x_i)^2\sigma_{x_i}^2$

**一阶系统**：$\tau=mC/(hA)$, $T(t)=T_F+(T_0-T_F)e^{-t/\tau}$, $E(t)=(T_\infty-T_0)e^{-t/\tau}$

**二阶系统**：$\omega_n=\sqrt{k/m}$, $\xi=\lambda/(2\sqrt{km})$, $K=1/k$

**频率响应**：$|G|=1/\sqrt{(1-n^2)^2+(2\xi n)^2}$, $n=\omega/\omega_n$

**可靠性**：$R=e^{-\lambda t}$, 串联 $R$ 相乘, 并联 $F$ 相乘

**电桥**：$U_o=E(\frac{R_1}{R_1+R_4}-\frac{R_2}{R_2+R_3})$, 1/4桥 $\frac{\Delta R}{4R}E$, 半桥 $\frac{\Delta R}{2R}E$, 全桥 $\frac{\Delta R}{R}E$

**应力应变**：$\sigma=F/A$, $\varepsilon_L=\sigma/E$, $\varepsilon_T=-\nu\varepsilon_L$, $\Delta R/R=G\varepsilon$

**电感**：$L=\mu_0 S W^2/(2\delta)$

**电容**：$C=\varepsilon_0\varepsilon_r A/d$

**噪声**：$W_N=\phi f_c$, SNR $=10\log_{10}(W_S/W_N)$, 滤波改善 $=10\log_{10}(B_{old}/B_{new})$, 平均改善 $=10\log_{10}(p)$

**加载效应**：$V_L=E_{Th}\cdot R_L/(R_{Th}+R_L)$

**加权平均**：$\bar{x}_w=\sum p_i x_i/\sum p_i$, $p_i=1/\sigma_i^2$

**最小二乘**：解 $\begin{cases}na+b\sum x=\sum y\\ a\sum x+b\sum x^2=\sum xy\end{cases}$

**交流电桥**：$Z_C=1/(j\omega C)$, 其余同直流电桥

**频率-数字**：$N_s/f_{sig}=N_c/f_{clk}$
